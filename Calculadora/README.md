# Calculadora Simples

Este é um projeto simples de calculadora desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Validação de campos vazios
- Validação para divisão por zero

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Objetivo do projeto
Este projeto foi criado para praticar:
- Estrutura HTML
- Estilização com CSS
- Manipulação do DOM
- Funções em JavaScript
- Lógica de programação

## Como executar
1. Baixe os arquivos
2. Abra o arquivo `index.html` no navegador
3. Digite dois números
4. Clique na operação desejada
